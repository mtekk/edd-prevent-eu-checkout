Prevents customer from being able to checkout until a minimum cart total is reached

== Description ==

This plugin requires [Easy Digital Downloads](http://wordpress.org/extend/plugins/easy-digital-downloads/ "Easy Digital Downloads"). 

This plugin does one simple thing. It will prevent a customer from being able to checkout if they're from the EU.
